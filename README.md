# ev3 is an idiomatic Go interface to an ev3dev device

The goal is to implement a simple Go style ev3dev API and helpers for common tasks.

Currently supported:

## Low level API

- [x] Automatic identification of attached devices
- [x] LED
- [x] LCD
- [x] Lego Port
- [x] Sensor
- [x] DC motor
- [x] Servo motor
- [x] Tacho motor
